# Lawless Context

Shared Lawless context for agents and teammates.

This repo is intentionally Markdown-first and public-readable for the current Lawless team workflow. Do not store secrets, env vars, private keys, client credentials, privileged client data, or long transcripts here.

## Structure

```text
index.md
people/
projects/
decisions/
meetings/
skills/
inbox/
```

Agents should read `index.md` first, then load only the relevant people, project, decision, meeting, or skill files.
